## 🚀 Projects:

### 📹 Video Text Extraction & Translation System
**Problem:** Extracting meaningful text from videos and making content accessible across multiple languages is time-consuming and requires manual effort.

**Solution:** Built a web-based automation tool that extracts text from video frames, translates it to multiple languages, and converts it to speech for better accessibility.

**Tech Stack:** 
- Backend: Python, Django
- Computer Vision: OpenCV, Tesseract OCR
- Translation: Google Translate API
- Text-to-Speech: GTTS (Google Text-to-Speech)
- Frontend: HTML, CSS, JavaScript
- Database: SQLite

**Key Features:**
- Automated OCR-based text extraction from video frames
- Multi-language translation support
- Audio generation with text-to-speech
- User-friendly web interface
